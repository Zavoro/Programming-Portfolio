This is my C# Interactive Systems Project

This project contains:
  Main Menu
  Number Guessing Game
  Dice Rolling Game
  Password Checker


This project showcases:
  Input handling
  Control flow
  Modular program structure
  Basic validation and error handling
  User friendly console UI design


Number Guessing Game:
  The program generates a random number between 1 and 10
  The player guesses until correct
  Feedback is given
  Includes input validation and a replay option

Dice Rolling Game:
  Simulates rolling a six‑sided die
  Allows repeated rolls
  Includes input validation and a return to menu option

Password Checker:
  Analyzes a password for:
    Uppercase letters
    Lowercase letters
    Digits
    Symbols
    Length
  Displays a color‑coded strength rating (Weak / Medium / Strong)
